# Filum Frontend

Phase A 前端骨架基于 **Vue 3 + TypeScript + Vite + Pinia + Vue Router + Element Plus**，并预留 Axios 作为统一 HTTP 客户端。

## 开发命令

```sh
npm install
npm run dev
```

## 验证命令

```sh
npm run test:unit
npm run build
npm run lint
```

## 当前骨架内容

- `src/router/`: 路由入口
- `src/stores/`: Pinia store
- `src/api/http.ts`: Axios 基础实例
- `src/views/HomeView.vue`: Phase A 占位首页
- `tests/`: 前端单元测试
